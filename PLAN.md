# 자산 포트폴리오 공유 서비스 기획안

## Context

부부 등 여러 사람이 함께 전체 자산 현황을 확인하고 투자 의사결정을 기록·추적·지원하는 웹 서비스.

- Phase 1: 2인(본인 + 아내) 프라이빗 서비스
- Phase 2: 공개 SaaS 서비스

개발 환경: 혼자 / 주 5h / 웹앱 (모바일 반응형)
기술 스택:

- Kotlin Spring Boot (Backend)
- Next.js TypeScript (Frontend)
- PostgreSQL

---

## 전체 기능 계획 (수정될 수 있음)

- 로그인
- 계좌, 자산 등 공동 관리 가능
- 계좌별, 종목별, 자산군 별 수익률 조회
- 자산 유형별 비중, 환 노출 비율, 연금자산 비율 등 자산 정보 조회
- 투자 의사결정 기록 (근거와 함께) 및 팔로업
    - 당시 매수/홀드/매도 했다면 예상 수익률
    - 일정 시간이 지난 후 회고 할 수 있도록 지원
- 포트폴리오를 등록 및 조회, 현재 상태와 목표 포트폴리오의 차이
    - 주기적 리밸런싱 알림, 혹은 비율 변동에 따른 리밸런싱 제안
    - 포트폴리오를 분석해서 위험하거나 조심해야 하는 부분 알려주는 기능
- 거장의 포트폴리오 조회, 내가 적용해볼 수 있는 기능
- 목표 포트폴리오에 가까워지도록 분할 매수/매도 계획 세워주기
- 연금 계좌 특별 처리
    - 세제 혜택 유형 표시 (퇴직금 / 연말정산 세액공제 대상 / 세액공제 대상 아님)
    - 즉시 출금 시 예상 세율
        - 퇴직금 및 세액공제 받은 금액 및 그 수익금은 기타소득세 16.5%
        - 세액공제 대상이 아닌 원금은 0%, 단 수익금은 마찬가지로 16.5%
        - 연금으로 인출하면 3.3%~5.5%
    - 세후 출금 가능 금액 계산
        - 세금 없이 인출 가능 금액
        - 기타 소득세 16.5% 납부 후 인출되는 금액
- 연간 성과 트래킹
    - 연도별 수익률 기록
        - 실질 수익률 = (연말 자산 - 연초 자산 - 순입금) / 연초 자산
    - 벤치마크 설정: 코스피 / S&P 500 / 나스닥 / 커스텀 (수동 입력 → 추후 API)
    - 벤치마크(베타 수익률) 대비 알파 수익률 표시
    - 연도별 성과 차트
    - 연도별 자산 규모 트래킹
